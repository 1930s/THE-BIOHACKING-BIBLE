[http://wishart.biology.ualberta.ca/PlasMapper/ Plasmapper] is a web based automated plasmid drawing and annotation tool which takes raw sequence data and recognizes common replication origins, antibiotic resistances, ORFs, and restriction sites. It produces a graphical or textual description of the annotation in PNG, JPG, or SVG form. [[Category:Protocol]] [[Category:In silico]] [[Category:DNA]]
