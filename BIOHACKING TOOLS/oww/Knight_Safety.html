Acetic acid can be stored with the flammables in a flammable cabinet without a secondary container. If Acetic acid is stored with inorganic acids, it should be in a separate secondary container since it is incompatible with inorganic acids. Trifluoracetic acid is also an organic acid so the same rules apply as with acetic acid. You do not need the acids stored in a closed [secondary] container.
