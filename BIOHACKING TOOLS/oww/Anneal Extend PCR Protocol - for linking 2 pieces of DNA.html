== Anneal Extend PCR == This is a protocol for linking any 2 pieces of DNA together using homologous primers. The original protocol can be found here[http://openwetware.org/images/9/95/Anneal_Extend.pdf]. As well, there is a protocol for introducing mutations via the primer. That protocol can be found here[http://openwetware.org/images/2/26/Megaprimer_PCR_Protocol.pdf]. [[Category:Protocol]] [[Category:In vitro]] [[Category:DNA]]
