Lining up the circle onto the ruler can be difficult and eye straining. =='''DATA'''== We took three sets of data which each contained ten measurements. For each set we found a value for the charge-to-mass ratio, which are : e/m = 6.28343E+11 +/- .725E+11 e/m = 3.39E+11 +/- .0272E+11 e/m = 3.07111E+11 +/- .00840E+11 '''Accepted Value of E/M''' e/m= 1.758 820 150(44)e+11 C/Kg *So as you can see we have a large discrepency between our experimental value and the accepted value.
