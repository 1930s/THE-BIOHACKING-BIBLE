Also see: [[Endy:Victor3 plate reader|Victor3 plate reader]] ==Introduction== This macro can be used to rearrange the results file from a [[Endy:Victor3 plate reader | plate reader]] experiment. The macro can be downloaded here for either a [[:image:PlateReader.xla | Mac]] or a [[:image:Plate_reader_windows.xla | PC]]. The macro creates a separate sheet for Absorbance and Fluorescence data. It then gives each well its own column on each sheet and puts the time points in the first column. It finally converts the time points to minutes format. The original version of this code was written by [[Heather Keller | Heather]]. There are further enhancements that could be made to the macro so please feel free to contribute to the code.
