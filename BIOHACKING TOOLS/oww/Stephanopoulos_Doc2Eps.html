Converting a figure in Microsoft Word to an Encapsulated Postcript file.

#Isolate the figure on its own page in the Word file. #Print (or convert) that page only to a PDF file. #Open the 1-page PDF file in Adobe Acrobat Professional. #From the menu, select Document -> Crop Pages. #Crop the page such that your figure has appropriate margins. #From the menu, select File -> Save As. #Save as an EPS file.
