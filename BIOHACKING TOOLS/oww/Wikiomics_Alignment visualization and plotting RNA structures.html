==RNA Alignment visualization== * [http://4sale.bioapps.biozentrum.uni-wuerzburg.de/index.html 4SALE] A tool for Synchronous RNA Sequence and Secondary Structure Alignment and Editing. * [http://www.biophys.uni-duesseldorf.de/construct3 ConStruct] Multipurpose tool for alignment editing, secondary structure prediction and visualizing RNA alignments (new version). * [http://www.sanger.ac.uk/Users/sgj/ralee/ RALEE] RNA ALignment Editor in Emacs. ==Plotting RNA structures== * [http://iubio.bio.indiana.edu/IUBio-Software+Data/molbio/loopdloop/java/loopDloop-doc.html loopDloop] a tool for drawing RNA secondary structures in molecular biology. * [http://jviz.research.iat.sfu.ca/ jViz.Rna] a Java application for visualizing and annotating RNA Secondary Structures * [http://iubio.bio.indiana.edu:7780/archive/00000377/ NAVIEW] A group of programs and scripts that produce plots of RNA secondary structures. (Link Broken) * [http://wilab.inha.ac.kr/pseudoviewer/ PseudoViewer] Automatically visualizing RNA pseudoknot structures as planar graphs. Needs Windows .NET Framework for "in web" use, else exe is available. * [http://www.rnadraw.com/ RNAdraw] an integrated program for RNA secondary structure calculation and analysis under 32-bit Microsoft Windows * [http://bioinfo.lifl.fr/carnac/rnafamily.html RNAfamily] A simple software tool that enables the display of all secondary structures for a family of RNA molecules. Uses a linear backbone representation. * [http://bibiserv.techfak.uni-bielefeld.de/rnamovies/ RNAmovies] a system for the visualization of RNA secondary structure spaces. * [http://www.tbi.univie.ac.at/~ivo/RNA/ RNAplot] Draw RNA Secondary Structures. Available formats are: PostScript (ps), Graph Meta Language (gml), Scalage Vector Graphics (svg), and XRNA save file (xrna). * [http://ndbserver.rutgers.edu/services/download/index.html RNAView/RnamlView] generates 2-dimensional displays of RNA/DNA secondary structures with tertiary interactions. * [http://rrna.uia.ac.be/rnaviz/ RnaViz] A user-friendly, portable, GUI program for producing publication-quality secondary structure drawings of RNA molecules. * [http://smi-web.stanford.edu/projects/helix/sstructview/home.html SStructView] a Java applet for viewing RNA secondary structures and linking to multiple computational backends. * [http://binkley.ncifcrf.gov/~bshapiro/structurelab/structureLab.html StructureLab] An integrated graphical software tool for predicting, plotting and analyzing RNA secondary structures. * [http://rna.ucsc.edu/rnacenter/xrna/xrna.html XRNA] a Java based suite of tools for the creation, annotation and display of RNA secondary structure diagrams.
