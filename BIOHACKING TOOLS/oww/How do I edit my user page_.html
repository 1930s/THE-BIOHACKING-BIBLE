To edit your userpage, you should first make sure that you have changed the abstract template correctly. You should change the line that says: '''[[Special:Mypage|your_full_name]] ''' so that it becomes: '''[[User:my_oww_user_name|My_Full_Name]]. ''' Where you have replaced '''my_oww_user_name''' with your OWW username, and '''My_Full_Name''' with your full name.
