='''Eiken Agar Plates'''= Eiken plates should be made up the same day or no more than one day before using them. Otherwise they dry out and cells do not swarm. ===Procedure (regular 10cm plates)=== # Add the following ingredients: #*0.25g tryptone #*0.20g NaCl #*0.05g eiken agar #*0.15g sodium citrate #*25ml water # Autoclave 20 min slow exhaust. # Add appropriate antibiotic # This gives a final [eiken] of 0.20% *Alternatively create a .28% Eiken Agar plate using ~.07g eiken agar ===Procedure (big 15cm plates)=== #Add the following ingredients: #*0.56g tryptone #*0.45g NaCl #*0.11g eiken agar #*0.338g sodium citrate #56.25 ml water # Autoclave 20 min slow exhaust. # Add appropriate antibiotic # gives a final [eiken] of 0.20% *This Alternatively create a .28% Eiken Agar plate using ~.1575g eiken agar [http://openwetware.org/wiki/IGEM:PennState Main]
