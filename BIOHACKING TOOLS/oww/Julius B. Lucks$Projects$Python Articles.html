I am a huge fan of the python programming language, and I think the python community has matured to the point that python can be a rich scientific programming platform, with the help of a few other tools. There is now solid support for data crunching through numeric modules, statistics through bindings to the r programming language, graphics and plot creation through matlab-like plotting modules and much much more. My goal is to write a series of articles to introduce python to scientists, and along with it, good programming practices that scientists can use to so that they can write quality code that makes use of state-of-the-art programming techniques.
