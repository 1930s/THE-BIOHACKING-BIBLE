Online tools:
[http://tools.neb.com/NEBcutter2/index.php ][http://tools.neb.com/NEBcutter2/index.php NebCutter V2.0] Restriction and other DNA analysis, with great GUI.
[http://frodo.wi.mit.edu/cgi-bin/primer3/primer3_www.cgi Primer3] Primer design tool best for creating compatible primer pairs for a given template.
[http://www.premierbiosoft.com/netprimer/netprlaunch/netprlaunch.html Netprimer] Primer test tool. Verify melting temperatures, hairpins, dimerization, &c of final primer designs.
[http://www.ncbi.nlm.nih.gov/ NCBI] National Center for Biotechnology Information
[http://www.ncbi.nlm.nih.gov/Entrez/index.html Entrez] Biology search engine, e.g. for looking up protein and DNA sequences by name
[http://www.ncbi.nlm.nih.gov/entrez/query.fcgi?db=PubMed PubMed] Literature search
[http://www.ncbi.nlm.nih.gov/BLAST/ BLAST] Basic Local Alignment Search Tool, for DNA/protein alignment and comparison, primer verification, etc.
[http://scholar.google.com/ Google Scholar] Most general literature search tool.

Freeware:
[http://www.ipw.agrl.ethz.ch/%7Elhennig/wingene.html WinGene] Tool for analyzing nucleotide sequences. Best for reverse complements and primer checks.
[http://www.umass.edu/microbio/mfi/ MFI] For converting flow cytometer output files to ASCII. Simple free registration.
[http://www.openoffice.org/ OpenOffice.org] Open source office environment, fully compatible (well, almost) with MS Office, including Powerpoint.
[http://open-perl-ide.sourceforge.net/ Open Perl IDE] Open source Integrated Development environment for Perl. Easy to set up and use.
[http://www.activestate.com/Products/ActivePerl/?_x=1 ActivePerl] Easy installation for running Perl scripts in Windows; necessary for Open Perl IDE.

Perl scripts:
[http://www.mit.edu/~mokelly/avo/deannotate_dna.pl deannotate_dna.pl] A useful script for annotating and editing DNA sequences. [http://www.mit.edu/~mokelly/avo/annotation%20documentation.txt This file] provides documentation and an example.
[http://www.mit.edu/~mokelly/avo/cytometry.zip cytometry.zip] Package of scripts and executables for quickly processing raw flow cytometer files into a table of histograms. Consult Michael on its use.
