ArrayBrowser

ArrayBrowser is developed to support livestock animal gene expression studies and eventually integrate gene expression data with physical and genetic data of different species. We are working towards the understanding of genes and pathways involved in the islet biology, and quality control of porcine islet cells prior to xeno-transplantation monitored by gene expression profiling with porcine specific genome chips, which contains more than 13K 70mer oligos. We are also developing bovine whole genome oligo chip to understand the genes/pathways involved in milk production. Those genome chips developed for livestock animals will allow more experiments designed to address the function of genes in process of medical and agricultural importance.

Sequencer Pipeline

These are a series of tools for coordinating the generation and database storage of DNA sequence data. This includes the automated generation of run sheets for the ABI3100 and the submission or editing of sequence data to our Comparative and Functional Genomics database.
GenBank submission Pipeline

This is a tool to assist in the submission of data from high volume DNA sequencing projects to public databases, including dbEST and dbSTS. Header files are prepared for individual or batch submission of sequences. Submission is coordinated with submission of trace files (TRACEdb) and user interactive masking, trimming and quality control.

Sequence Annotation Pipeline.

This is a tool to assist in the annotation of sequence data.

Chromat Analyzer

This tool permits secure viewing of chromatograms and FASTA sequences from remote locations.  You can view data already resident on our server, or upload data from remote locations. Please use Internet Explorer or Netscape 6.2 web browser for this tool.
Primer3

Primer3 selects primers for polymerase chain reaction (PCR), hybridization and DNA sequencing using sequence provided by the user.  If desired, sequence quality can be considered in the selection of primers. Target sequence can be provided individually or in batch as either Chromatograms OR FASTA files.
Overgo4.0

Overgo 4.0 selects Overgo oligos for the synthesis of high specific activity hybridization probes. Target sequence can be provided individually or in batch as either Chromatograms OR FASTA files.

Sequence Concatenation Tool

This program allows for the concatenation of two sequences, and their associated quality scores, directly from chromatograms.  The user has the option to reverse complement either sequence before concatenation.  Virtual unsequenced stuffer fragments of selectable length can be inserted between the concatenated sequences.  This software is particularly valuable for creating sequence to be used in the design of PCR primers that span unsequenced gaps. If you wish to concatenate a large collection of sequences (batch) please contact system administrator.
SeqCart

This allows user to freely group sequences into different groups according to various analysis tasks. These groups can be used in later calculations such as primer design, sequence submission, Blast search, and data mining.

OrthoMap

This tool is used to identify sequence orthologs and their position in reference genomes. FASTA files can be uploaded or trace files can be selected from our server via SeqCart. Based on parameters set by the user, sequences are masked and compared to the sequence of selected vertebrate genomes currated at ENSEMBL. Sequences are hierarchically compared using WU-BLASTX to identify orthologous proteins, WU-BLASTN to identify orthologous cDNAs, and finally WU-BLASTN to identify orthologous intergenic DNA. Eventually, this tool will permit PCR primer design that makes use of information about orthologous gene-structure.
BLAST

DNA Sequences can be selected from our server for comparision to public data repositories based on the algorithm developed by Altschul et al. (1990). Masks for submission and reporting defined by the user.
BLAT
This tool was developed by Jim Kent and can be used to quickly find regions of the human or mouse genomes with high similarity to your sequence (95% and greater similarity over 40 bases or more). DNA BLAT works by keeping an index of the entire human OR mouse genomes in memory.  Coming soon, local BLAT of individual and batches of sequences selectable from our server or loaded from a remote location.
PolyView

This tool was developed for annotating and summarizing the results from comparative re-sequencing projects (for SNP discovery) based on the PHRED/POLYPHRED/CONSED suite.  Sequences can be submitted for analysis from our server or from remote locations.  In order to annotate or edit contigs requires the implementation of X-windows.
ChromatAnnotator

This is a tool based on the PHRED/POLYPHRED/CONSED suite and allows the user to assemble and annotate sequence contigs.  Sequences on our server or loaded from remote locations can be analyzed.  Annotation can be directly parsed to our database for later query.  In order to annotate or edit contigs requires the implementation of X-windows.
Pairwise SEQALIGN

This tool allows the pairwise comparison of chromatograms to reference chromatograms or GenBank entries. Individual or batches of sequences can be selected from our server or loaded from a remote location.  If you want to conduct a pairwise comparison of a large collection of sequences (batch) contact the system administrator.

Locusmap and Pedigraph

 (These tools are provided by Dr. Yang,  Da, Department of Animal Science, U of MN.)

Locusmap is a computer package designed for rapid linkage analysis and map construction using a variety of inheritance modes. Pedigraph permits pedigree visualization for studying population structure and inheritance patterns.
Comparative and Functional Genomics Database Schema

A schema from the USDA Meat Animal Research Center has been developed for dealing with genetic and DNA sequence data in a species independent manner.  We are collaborating with the USDA to modernize and distribute this schema.
